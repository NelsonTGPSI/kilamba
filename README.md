# Cantinho do Kilamba Conforto - Casa de Curta Duração

Uma casa de curta duração localizada na Cidade do Kilamba, condomínio Kilamba conforto, oferecendo 6 quartos suites, piscina, bar, quintal, internet e climatização total.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
